# Travel Agency — современный сайт туристического агентства

Адаптивный лендинг туристического агентства. Проект демонстрирует навыки вёрстки и работы с пользовательскими интерфейсами.

## 🛠 Стек технологий

- **React 18** (функциональные компоненты, хуки)
- **TypeScript** (типизация пропсов и состояний)
- **SCSS** (модульная стилизация, адаптив)
- **Vite** (быстрая сборка и горячая перезагрузка)

## 📦 Функционал

- Полностью адаптивная вёрстка (мобильные устройства, планшеты, десктоп)
- Интерактивные элементы (формы, карусели, всплывающие окна — если они есть в проекте)
- Оптимизация производительности и загрузки изображений

## 🚀 Запуск проекта

```bash
# Клонируйте репозиторий
git clone https://github.com/DenisCalenici/Travel-Agency.git

# Перейдите в папку проекта
cd Travel-Agency

# Установите зависимости
npm install

# Запустите сервер для разработки
npm run dev
После запуска проект будет доступен по адресу http://localhost:5173.

📁 Структура проекта (пример)
text
src/
├── components/    # Переиспользуемые компоненты (карточки, формы, шапка, подвал)
