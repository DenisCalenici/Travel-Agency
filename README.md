
# ✈️ Travel Agency — современный сайт туристического агентства

Адаптивный лендинг туристического агентства. Проект демонстрирует навыки вёрстки и работы с пользовательскими интерфейсами.

---

## 🛠️ Стек технологий

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white)
![SCSS](https://img.shields.io/badge/SCSS-CC6699?style=for-the-badge&logo=sass&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)

- **React 18** — функциональные компоненты, хуки
- **TypeScript** — типизация пропсов и состояний
- **SCSS** — модульная стилизация, адаптив
- **Vite** — быстрая сборка и горячая перезагрузка

---

## 📦 Функционал

- 📱 Полностью адаптивная вёрстка (мобильные устройства, планшеты, десктоп)
- 🎨 Интерактивные элементы (формы, карусели, модальные окна)
- ⚡ Оптимизация производительности и загрузки изображений
